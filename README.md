# BD Speed Optimizer

WordPress performance plugin with speed scoring, frontend optimizations, and database cleanup.

## Features

- **Performance Scanner** — 14-point check with 0-100 score
- **Defer JavaScript** — Move JS loading to improve page speed
- **Delay JavaScript** — Delay non-critical JS until user interaction
- **Remove jQuery Migrate** — Eliminate unnecessary legacy script
- **Minify HTML** — Strip whitespace and comments from output
- **Lazy Load** — Images and iframes loaded on scroll
- **Disable Emojis/Embeds** — Remove WordPress emoji and embed scripts
- **DNS Prefetch & Preconnect** — Resource hints for faster loading
- **Database Cleanup** — Revisions, spam comments, expired transients

## Requirements

- WordPress 5.8+
- PHP 7.4+
